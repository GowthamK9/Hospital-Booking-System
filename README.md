> "Doctor Appointment Slot Booking System"

A full-stack basic web application where admins create doctors & slots, and users view and book available appointment slots.

> Overview

This project is a complete appointment scheduling system built using:

React + Vite (Frontend)

Node.js + Express (Backend)

PostgreSQL (Database)

Users can see available slots, while admins can create doctors and define available time slots.

> Admin Features

Create new doctor profiles

Add available time slots

Manage appointments

> User Features

View available doctors and time slots

Click and book a slot

View individual slot details

> Technical Features

REST API architecture

PostgreSQL relational database

Environment variable support

Clean component-based React UI
